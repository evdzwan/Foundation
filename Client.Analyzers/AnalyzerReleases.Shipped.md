## Release 0.1

### New Rules

Rule ID | Category | Severity | Notes
--------|----------|----------|------------------------------------------------------------------------
FNDD001 | Usage    | Error    | Effects must have exactly 2 parameters
FNDD002 | Usage    | Error    | Effects must have return void or Task
FNDD003 | Usage    | Error    | Parameter must be of type IDispatcher
FNDD004 | Usage    | Error    | Reducers must have exactly 2 parameters
FNDD005 | Usage    | Error    | Reducers must have a return type that matches the first parameter type
